# automatick

> Boilerplate components to create a model simulation with React

[![NPM](https://img.shields.io/npm/v/automatick.svg)](https://www.npmjs.com/package/automatick) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com) ![test](https://github.com/jckr/automatick/workflows/test/badge.svg)

## Install

```bash
npm install --save automatick
```

## Usage

See [documentation](https://jckr.github.io/automatick/) for [guide](https://jckr.github.io/automatick/tutorial) and [examples](https://jckr.github.io/automatick/examples)

```jsx
import React from 'react'

import {Model} from 'automatick'

class Example extends Component {
  render() {
    return <Model />
  }
}
```
