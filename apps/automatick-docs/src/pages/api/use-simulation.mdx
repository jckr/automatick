# useSimulation

```ts
import { useSimulation } from 'automatick/react/hooks';
```

Hook that reads simulation state and actions from the nearest `<Simulation>` provider. Throws if used outside a `<Simulation>`.

```ts
function useSimulation<T = unknown, FallbackParams = unknown>(): SimulationContextValue<...>
```

**Type inference:** pass `typeof mySim` as the first type parameter to infer `Data` and `Params`, or provide them explicitly.

---

## Return value

### State

| Field | Type | Description |
|-------|------|-------------|
| `data` | `Data` | Current simulation state. |
| `params` | `Params` | Current parameters. |
| `tick` | `number` | Current tick number. |
| `status` | `SimulationStatus` | Current engine status. |

### Actions

| Field | Type | Description |
|-------|------|-------------|
| `play` | `() => void` | Start or resume the simulation. |
| `pause` | `() => void` | Pause a playing simulation. |
| `stop` | `() => void` | Permanently stop the simulation. |
| `seek` | `(tick: number) => void` | Jump forward to a target tick (pauses the sim). |
| `advance` | `(count?: number) => void` | Step forward by `count` ticks (default `1`). Pauses if playing. |
| `setParams` | `(patch: Partial<Params>) => void` | Merge a partial update into current params. |
| `resetWith` | `(patch?: Partial<Params>) => void` | Re-initialize state from `init()`. Optionally merge a param patch first. Resets tick to `0` and status to `idle`. |

---

## SimulationStatus

| Value | Description |
|-------|-------------|
| `'idle'` | Engine created but never played, or just reset. |
| `'playing'` | Actively ticking. |
| `'paused'` | Paused mid-run. Can resume or step. |
| `'stopped'` | Terminated (by `shouldStop`, `maxTime`, or manual `stop()`). Cannot resume. |

---

## Example

```tsx
import { useSimulation } from 'automatick/react/hooks';
import counterSim from './counterSim';

function CounterDisplay() {
  const { data, tick, status, play, pause } = useSimulation<typeof counterSim>();

  return (
    <div>
      <p>Tick {tick}: count = {data.count}</p>
      <p>Status: {status}</p>
      <button onClick={status === 'playing' ? pause : play}>
        {status === 'playing' ? 'Pause' : 'Play'}
      </button>
    </div>
  );
}
```

# useSimulationCanvas

```ts
import { useSimulationCanvas } from 'automatick/react/canvas';
```

Hook for canvas-based rendering that bypasses React's render cycle. Subscribes to the engine directly and calls the draw callback imperatively on every snapshot.

```ts
function useSimulationCanvas<T = unknown, FallbackParams = unknown>(
  draw: (ctx: CanvasRenderingContext2D, snapshot: { data: Data; params: Params; tick: number }) => void
): React.RefObject<HTMLCanvasElement | null>
```

Returns a ref to attach to a `<canvas>` element.

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `draw` | `(ctx, snapshot) => void` | Called on every engine snapshot. Receives the 2D context and `{ data, params, tick }`. |

## Example

```tsx
import { useSimulationCanvas } from 'automatick/react/canvas';
import boidsSim from './boidsSim';

function BoidsCanvas() {
  const canvasRef = useSimulationCanvas<typeof boidsSim>((ctx, { data, params }) => {
    ctx.clearRect(0, 0, 400, 400);
    data.forEach((boid) => {
      ctx.beginPath();
      ctx.arc(boid.x, boid.y, 3, 0, Math.PI * 2);
      ctx.fill();
    });
  });
  return <canvas ref={canvasRef} width={400} height={400} />;
}
```
